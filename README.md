# 🚀 JavaScript_by_Iker

> Aprende JavaScript desde 0 hasta 100. Sin atajos. Sin humo. Entendiendo lo que realmente importa.

---

## 📌 Descripción

**JavaScript_by_Iker** es un repositorio diseñado para enseñarte JavaScript desde los fundamentos más básicos hasta los conceptos más avanzados y técnicos.

No se trata solo de escribir código que "funcione".
Se trata de entender qué está pasando por dentro y desarrollar mentalidad de programador.

Este proyecto está pensado tanto para:

* 🟢 Personas que empiezan desde cero
* 🟡 Desarrolladores junior que quieren reforzar bases
* 🔵 Programadores que quieren entender mejor cómo funciona JavaScript internamente

---

## 🧠 Filosofía de aprendizaje

Aquí no memorizarás código.
Aprenderás a:

* Pensar en términos de lógica
* Entender el flujo de ejecución
* Comprender cómo funciona el motor de JavaScript
* Identificar errores y razonar soluciones

El objetivo es que puedas aplicar lo aprendido en proyectos reales.

## 💻 Cómo usar este repositorio

1. Clona el repositorio:

```bash
git clone https://github.com/tuusuario/JavaScript_by_Iker.git
```

2. Accede al proyecto:

```bash
cd JavaScript_by_Iker
```

3. Recorre las carpetas en orden.
4. Lee las explicaciones.
5. Ejecuta el código.
6. Modifícalo y experimenta.

💡 Consejo: romper el código y arreglarlo es parte del aprendizaje.
